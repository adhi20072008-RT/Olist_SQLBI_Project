# Olist E-Commerce Sales & Customer Analytics Project

This project presents an end-to-end SQL and Power BI data analytics workflow using the Olist Brazilian E-Commerce dataset.
The analysis focuses on customer behavior, sales performance, revenue trends, and RFM segmentation.

## Tools & Technologies Used

* MySQL
* Power BI
* SQL Window Functions
* Data Cleaning & Transformation
* KPI Dashboarding
* RFM Customer Segmentation

## Key Features

* Sales & Revenue Analysis Dashboard
* Customer Analytics Dashboard
* RFM Segmentation Analysis
* Customer Ranking using SQL Window Functions
* Interactive Power BI Visualizations
* Business Insights & KPI Tracking
